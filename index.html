Cavecrawler Mapper
Made by captivator and fizzy
for Lumber Tycoon 2 players with love ❤️
Hosted on GitHub
Every 4 days, the maze changes the path to cavecrawler wood, so with this tool you can find the working map every 4 days.
Note: the path to cavecrawler is decided during server creation, so if you are playing on an old server, current path may not work.
